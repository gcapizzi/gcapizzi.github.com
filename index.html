Cosa so fare

Progettazione e programmazione orientata agli oggetti
Ho un'ottima padronanza dei principi della progettazione orientata agli oggetti. Punto sempre alla costruzione di soluzioni flessibili, riusabili e manutenibili applicando i design pattern e i principi SOLID.
Java
Conosco molto bene la piattaforma Java SE, che ho utilizzato durante tutto il corso dei miei studi. In questi ho avuto occasione di approfondire anche gli aspetti più complessi del linguaggio e di lavorare a diversi progetti universitari che ne facevano uso.
PHP
PHP è stato il linguaggio con cui per primo mi sono avvicinato allo sviluppo web. Lo utilizzo da svariati anni e lo conosco in modo approfondito, soprattutto nei suoi aspetti object oriented. Attualmente lo utilizzo soprattutto accompagnato da framework come CakePHP o Symfony.
Ruby e Ruby on Rails
Ruby è senza dubbio uno dei miei linguaggi preferiti. Nel tempo ho avuto modo di studiarne tutti gli aspetti, compresi quelli legati alla metaprogrammazione. Il framework Ruby on Rails mi ha insegnato un nuovo modo di sviluppare per il web ed è spesso la mia prima scelta per lo sviluppo di web app basate sul paradigma REST.
Python
Ho una buona padronanza di Python e lo utilizzo spesso per realizzare script di vario genere. Uno di questi script si è evoluto in una vera e propria applicazione, FreiTag, nata dall'esigenza di taggare e rinominare in modo automatico grandi quantità di file MP3. Grazie ad essa ho avuto modo di approfondire tutti gli aspetti del linguaggio, dall'OOP al testing.
Controllo di versione
Il codice di ciascuno dei miei progetti è gestito da un sistema di controllo di versione. Dopo avere scoperto il mondo dei VCS grazie a Subversion mi sono convertito ai sistemi distribuiti, come Mercurial ma soprattutto Git.
HTML e CSS
Ho scritto HTML e CSS per un gran numero di siti web. Sono in grado di ottenere layout grafici anche complessi mantenendo il markup il più possibile pulito e fedele alla semantica. Utilizzo tecniche quali il progressive enhancement e la graceful degradation per fornire a tutti gli utenti l'esperienza migliore pur supportando i browser meno aggiornati. Per avere un'idea di come scrivo in HTML e CSS, potete visualizzare il sorgente di questa stessa pagina, o quello dei siti citati nella sezione Cosa ho fatto.
JavaScript e jQuery
Utilizzo JavaScript, sempre in combinazione con framework come jQuery, per realizzare siti web dinamici ed animati come questo. Per dimostrare che è possibile utilizzare gli standard del web anziché tecnologie proprietarie come Flash, ho realizzato un clone del menu animato del sito dell'azienda di trasporti pubblici della mia città. Il progetto si chiama atibi ed è visibile qui.
WordPress e Drupal
Ho utilizzato WordPress sin dalla versione 2.0 per realizzare diversi siti web a livello professionale. Tra questi il sito del centro di ricerca CYFE e del Bergamo Linux User Group. Mi sono inoltre occupato dello sviluppo frontend dei siti del Gruppo Pigna, basati sulla piattaforma Drupal.
Unix
Sono utente Linux dal 2003 e utente Mac OS X dal 2009. Sono membro del direttivo del BgLUG (Bergamo Linux User Group), il Linux User Group della mia città, di cui sono socio dal 2006. Sono completamente a mio agio con la riga di comando, e sono in grado di svolgere le principali operazioni di gestione di questo tipo di sistemi.
